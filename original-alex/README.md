# Vision Data Models

## Getting Started

### Dependencies

- [SPDLogger](https://gitlab.com/oasys/third-party/common)
_Optional_
- [Protobuf](https://gitlab.com/oasys/third-party/protobuf)
- [GStreamer (Optional)](https://gitlab.com/oasys/third-party/gstreamer)

### Building

```bash
mkdir -p build
cd build
cmake ..
make -j$(nproc)
sudo make install
```
